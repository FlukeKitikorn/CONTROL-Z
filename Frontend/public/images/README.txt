โฟลเดอร์รูปสแตติก — อ้างอิงจากเบราว์เซอร์เป็น  /images/...

โครงแนะนำ
  auth/       หน้าเข้าสู่ระบบ / ลงทะเบียน (พื้นหลัง, โลโก้)
  brand/      โลโก้หรือภาพแบรนด์ใช้ทั่วแอป (navbar, favicon แบบ png ฯลฯ)
  ui/         ภาพประกอบทั่วไป (illustration, empty state)

ไฟล์ที่โค้ดอ้างอยู่แล้ว
  /images/bg-auth.jpg   — พื้นหลังฝั่งซ้าย login/register
  /images/bg-logo.png   — โลโก้ในกล่องกลางคอลัมน์ซ้าย (แก้ path ใน AuthSplitLayout ถ้าใช้ชื่ออื่น)
